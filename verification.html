Verification & review workflow

Precise factual claims should point inline to an authoritative source. A ⚠️VERIFY tag means the claim is intentionally not trusted yet and must also appear in the exact-item table on this page.

Review standard

  1. Open the cited primary source; do not verify a specification claim from a blog summary.
  2. Check that the source supports the exact wording, including version, target platform, units, preconditions, and exceptions.
  3. If it does, replace the verify tag with a linked src pointer.
  4. If it does not, correct or remove the claim. Never retain a useful-sounding number merely because it is plausible.
  5. For repository internals, record commit/tag plus exact file and function. A branch name alone is not a reproducible source.
Self-contained does not mean unsourced

The lesson must teach the concept, assumptions, and consequences in its own text. The source link is evidence and optional depth. “Read this external chapter to understand the lesson” is not acceptable; neither is an unsupported claim with no source.

When authoring Phases 2–8

  • Add an exact table row before publishing any unresolved byte offset, field size, lock name, algorithm step, source symbol, benchmark, or lecture timestamp.
  • For SQLite format claims, cite the relevant official format section inline.
  • For Isar, LMDB, or libmdbx internals, pin the version/commit and do not infer one version's architecture from another.
  • For APIs and tools, pin the dependency used by the exercise. “Current latest” is not a stable curriculum contract.
Examples of corrected claims
  • slice::from_raw_parts(null, 0) is not allowed; map the FFI empty pair to &[] without calling it.
  • An escaping Rust panic at extern "C" aborts; C-unwind is the unwind-permitting ABI. A foreign exception entering a non-unwind Rust ABI is undefined behavior.
  • sync_all() is an OS synchronization request, not an unconditional statement about every filesystem, device cache, power-loss mode, or directory update.
  • Copy-on-write is a mechanism used by some crash-safety and MVCC designs, not those guarantees “for free.”
