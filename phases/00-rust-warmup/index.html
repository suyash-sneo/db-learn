Phase 0

Just-enough Rust + re-anchor

You can't build a database in a language you can't read. This phase gives you the exact slice of Rust the engine needs — ownership, errors, traits, byte buffers, files, serde — taught from a beginner's footing, with every feature introduced the first time it appears. It ends with a from-scratch key-value store: your Rust litmus test and a deliberate reset of the AI-dependence habit.

What you'll be able to do after Phase 0

Read and write idiomatic Rust comfortably enough to implement a pager and a B+tree: move values around without fighting the borrow checker, model data with structs/enums, handle errors with Result and ?, push bytes through &[u8]/Vec<u8>, and read/write files with explicit seeks.
